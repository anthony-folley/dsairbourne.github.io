Project Magpie

Project Magpie, AirBourne's revolutionary Remote Desktop Software, is poised to redefine the digital workspace experience. With key features such as on-device low-power rendering, high-security transfers utilizing AirBourne's RaE Security System, and a non-rigid release structure, Project Magpie prioritizes user-centric innovation. As it approaches the end of its beta phase, this cutting-edge software is preparing for a seamless transition to Dis//Connect. Beyond a mere name change, this evolution encompasses enhanced connectivity, innovative disconnection features, and an intuitive interface. Dis//Connect signifies the next phase in remote desktop experiences, where users can seamlessly navigate a customized workspace, even in offline mode. With a commitment to quality assured by beta testers and a vision that places users at its core, Project Magpie's transition to Dis//Connect promises to usher in a transformative era of unparalleled digital collaboration and access.

Key Features of Project Magpie

  • On Device Low Power Rendering: Enjoy efficient rendering on your device, conserving power while ensuring a seamless remote desktop experience.
  • New Rendering System: Project Magpie introduces an advanced rendering system, enhancing the visual quality and overall performance for an immersive user experience.
  • High Security Transfer with RaE Security System: Your data is safeguarded with Project Magpie's high-security transfer, incorporating AirBourne's robust RaE Security System, providing a shield against potential threats in the digital realm.
  • Multi-Monitor Support: Whether your host has multiple monitors or not, Project Magpie adapts. Configure it to utilize all your monitors, providing an expansive workspace for improved productivity.
  • Non-Rigid Release Structure: Breaking free from traditional release schedules, Project Magpie follows a non-rigid structure, prioritizing quality and innovation over fixed timelines.
  • Free: Project Magpie is not just powerful; it's accessible to everyone. Enjoy the benefits of this advanced remote desktop software without any cost.
  • Offline Mode: Stay productive even when disconnected. Project Magpie's offline mode allows you to perform tasks in "low power" or, if desired, run "high power" applications on your local machine/device.
  • Custom Workspace for User Interaction: Tailor your digital workspace with Project Magpie. Apply skins, mods, and more to create a personalized environment that aligns with your preferences.
  • Quality Assured by Beta Testers: Project Magpie's quality is not left to chance. Beta testers actively contribute to the refinement process, ensuring that each release meets the highest standards of functionality and user satisfaction.
  • Made For All: Project Magpie is designed to cater to a diverse user base. Whether you're an individual user or part of a large enterprise, the software is made for all, adapting to varying needs seamlessly.
  • Users at the Heart: Placing users at the core of its design philosophy, Project Magpie ensures that the software evolves based on user feedback, preferences, and insights. The user experience is prioritized to create a truly user-centric remote desktop solution.
